GPT 공작소 Lite v1.1
====================

v1.1 변경 내용 (v1.0 대비)
- GUI 한 화면 통합: 업로드 / 자료실 / 미리보기 / Blogger 저장을 화면 전환(탭) 없이
  한 화면에서 동시에 다루는 통합 작업대로 재구성했습니다.
  - PC(가로 폭 860px 이상): 좌(업로드+자료실) - 중(미리보기) - 우(Blogger 저장) 3열 패널
  - 모바일: 세로 카드형으로 위→아래 배치
  - 전체 페이지는 스크롤되지 않으며, 각 패널 내부(자료실 목록/미리보기/오류 목록/
    진행 상태)만 스크롤됩니다.
- 아이콘 추가: 외부 아이콘 라이브러리 없이 이모지로 각 영역을 표시합니다.
  (📦 업로드, 📚 자료실, 👁 미리보기, 📝 Blogger 저장, 🔐 로그인, ✅ 완료, ⚠️ 주의, 🔄 처리중)
- Blogger 표 반영 문제 개선: Blogger 저장용 본문 HTML에서 3열 이상 table을
  div/h3/p 기반 카드형 목록으로 변환합니다(2열 이하 table은 그대로 유지).
  또한 원본 인라인 font-size를 제거해 Blogger 모바일 화면에서 글씨가 과도하게
  크거나 뭉개지는 문제를 줄였습니다. 이 변환은 Blogger 전송용 HTML에만 적용되며,
  자료실 미리보기 화면은 원본 표 그대로 보여줍니다.
- 모바일 가독성 개선: 통합 작업대 배치와 함께 카드/버튼 여백을 정리했습니다.
- 코드 정리: 어디서도 호출되지 않던 worker-api-module.js의 checkBloggerStatus/
  checkWorkerHealth, 탭 전환용 CSS, 상세보기 팝업용 CSS(더 이상 쓰이지 않음)를
  제거했습니다. Gemini/품질검수/점수/SEO/예약/통계/백업 관련 코드는 v1.0부터
  포함된 적이 없습니다.

건드리지 않은 핵심 로직 (v1.0과 동일)

- R2 이미지 업로드 (r2-image-module.js) — 수정 없음
- 자료실 저장/불러오기 (library-module.js, storage-module.js) — 수정 없음
- ZIP 파싱 핵심 흐름 (upload-module.js) — 수정 없음
- 이미지 경로 치환 흐름 (blogger-save-module.js의 collectImageTasks/이미지
  R2 업로드/최종 img src 교체 로직) — 수정 없음. table/font-size 처리는 이 로직
  "이전"에 실행되는 별도 전처리 단계로 추가했습니다.
- Worker 주소(https://wooow.qudrnr84.workers.dev), R2 공개 URL 등 — 수정 없음

설치 방법
- 이 ZIP의 압축을 풀면 나오는 index.html / css / js / version.json 을 그대로
  GitHub Pages 저장소 루트에 덮어씁니다(상위 폴더를 한 번 더 만들지 않습니다).
- v1.0 파일을 미리 백업해두는 것을 권장합니다.

실기 테스트 필요 항목
1. 로그인 → 통합 작업대 진입 확인
2. PC 브라우저에서 3열 패널 배치, 모바일(또는 브라우저 폭 축소)에서 세로 배치 확인
3. 전체 페이지 스크롤 없이 각 패널 내부만 스크롤되는지 확인
4. ZIP 업로드 → 체크리스트/미리보기 패널 표시 확인 → 자료실 저장
5. 자료실에서 글 선택 → 미리보기/Blogger 저장 패널에 동시에 반영되는지 확인
6. Blogger 임시저장 실행 → 표가 있던 자료가 Blogger 관리자 화면에서 카드형으로
   정상 표시되는지, 글씨 크기가 적절한지 확인
7. 이미지가 기존과 동일하게 정상 반영되는지 확인(v1.0에서 이미 확인된 부분이나
   재확인 권장)
