                    다음 업데이트 후보: 오류 코드 체계 확장, 게시판 정렬/검색 옵션, 저장 전 HTML 정리 옵션, 화면 표시 옵션.
                    
          
                    개발 메모: ZIP 업로드 후 성공한 글은 확인 없이 바로 게시판에 자동 저장되고, 결과는 성공/실패 개수만 보여주는 고정 카드 하나로 안내됩니다. 게시판은 9개 카테고리를 한 줄씩 보여주는 고정 목록 + 선택된 글 정보창(제목/짧은 요약/[미리보기]/[선정자료 보기])으로 재구성됐습니다. 네이버 복사/블로그 임시저장은 각자의 영역(네이버·블로그스팟)에서만 실행합니다. 신규 기능은 모두 선택형 모듈로 분리되어 있습니다. 특정 기능에 문제가 있으면 해당 모듈만 제거해도 기본 업로드/저장 흐름은 계속 사용할 수 있습니다.
